# **App Name**: AdIntel

## Core Features:

- User Registration: Allow influencers to register their profiles, including name, contact details, advertised products (images and text).
- Image and Product Upload: Enable users to upload images of themselves and details of the products they promote.
- AI-Powered Verification: Search for YouTube videos using product information, and employ a tool that scans the video and identifies faces to determine correct advertisement of products.
- Verification Result Display: Display verification results to the user, indicating whether the correct face and products were found in the analyzed videos.
- Dashboard: Display basic user profile information, uploaded products, verification results, and alerts on a unified dashboard.

## Style Guidelines:

- Primary color: HSL(210, 67%, 46%) -> RGB(#388bf2), a classic, versatile blue tone suitable for user interfaces
- Background color: HSL(210, 20%, 95%) -> RGB(#F0F4FF), a desaturated tint of blue that creates a calm and clean surface
- Accent color: HSL(180, 67%, 46%) -> RGB(#38f2e7), a vibrant cyan color, different in both brightness and saturation, used to highlight key elements and actions
- Body and headline font: 'Inter', a grotesque-style sans-serif font, known for its readability and modern appearance
- Use a consistent style of filled icons with rounded corners to match the modern design.
- Employ a grid-based layout to ensure a consistent and responsive design across different devices.
